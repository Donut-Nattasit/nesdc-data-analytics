# Visualization Templates Skill

## Workflow
1. **Tool Selection**: Use Altair for interactive web charts (limit 5k rows) and Matplotlib for static report charts.
2. **Formatting**: Always provide clear titles, axis labels, and legends using the templates in [chart_templates.md](references/chart_templates.md).
3. **Thai Localization**: If "Thai" is requested, strictly follow the font and year conversion guidelines in the reference.
4. **Export**: Save all charts as PNG to `output/chart/` with high DPI (300).

## Reference Materials
- [chart_templates.md](references/chart_templates.md): Standardized code for line charts, stacked bars, and Thai localization.
